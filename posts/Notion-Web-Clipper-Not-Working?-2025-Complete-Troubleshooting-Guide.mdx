## Common Error Messages & Fixes

### "Extension context invalidated"

**What it means:** Extension was updated or reloaded while you had pages open.

**Fix (30 seconds):**
1. Close all Chrome tabs
2. Restart Chrome completely
3. Open fresh tabs

**Prevention:** Don't reload extensions while actively using them.

### "Failed to save to Notion"

**What it means:** Connection issue with Notion's servers or API.

**Fix attempt 1 (1 minute):**
1. Check Notion's status page: status.notion.so
2. If down, wait and retry
3. If up, proceed to fix attempt 2

**Fix attempt 2 (2 minutes):**
1. Log out of Notion completely (notion.so)
2. Close browser
3. Restart browser
4. Log back into Notion
5. Retry save

**Fix attempt 3 (3 minutes):**
1. Remove extension
2. Reinstall from Chrome Web Store
3. Reconnect to Notion
4. Test save

### "Permission denied" or "Access blocked"

**What it means:** Website's Content Security Policy blocks extension.

**Sites this commonly affects:**
- GitHub (sometimes)
- Enterprise tools behind SSO
- Government websites
- Banking sites

**Workarounds:**
1. Copy content manually to temporary tab
2. Save from the copied content
3. Or screenshot â†’ OCR â†’ save text

**Note:** This is a security feature, not a bug. Can't be "fixed."

### "No pages available"

**What it means:** Extension can't see your Notion workspace.

**Fix (2 minutes):**
1. Click extension icon
2. Check workspace selector
3. If wrong workspace showing:
   - Disconnect account
   - Reconnect and authorize correct workspace
4. If no workspaces showing:
   - Sign out of Notion
   - Sign back in
   - Reauthorize extension

### Right-click menu option missing

**What it means:** Context menu isn't registered properly.

**Fix (1 minute):**
1. Chrome Extensions page
2. Toggle extension OFF then ON
3. Refresh current page
4. Right-click again

**If still missing:**
1. Remove extension
2. Restart Chrome
3. Reinstall extension
4. Refresh pages

## Permission Issues

### Extension Not Accessing Pages

**Issue:** "Click to enable on this site"

**What it means:** Extensions defaulted to click-to-run mode.

**Fix (permanent):**
1. Right-click extension icon
2. "This can read and change site data"
3. Select "On all sites" or "On [specific site]"

**Issue:** "Extension cannot access this page"

**What it means:** Chrome Web Store, settings, or restricted pages.

**Reality check:** Extensions can't work on:
- chrome:// URLs
- Chrome Web Store pages
- New Tab page
- Some Google services

**Workaround:** Copy content to temporary document, save from there.

## Data & Sync Issues

### Saves Not Appearing in Notion

**Check these:**

1. **Right workspace?**
   - Check workspace selector in extension
   - You might have multiple Notion accounts

2. **Right page/database?**
   - Verify target page in extension settings
   - Search Notion for the save (might be in unexpected location)

3. **Sync delay?**
   - Wait 30 seconds, refresh Notion
   - Check "All Updates" in Notion sidebar

4. **Quota exceeded?**
   - Free tier: 30 saves/month
   - Check usage in extension popup
   - Upgrade or wait for next month

### Duplicate Saves

**Issue:** Same content saved multiple times

**Cause:** Clicking save button repeatedly during lag.

**Fix:**
- Wait for confirmation before clicking again
- Check "Saving..." indicator
- Use CMD+Z / CTRL+Z to undo duplicates in Notion

**Prevention:**
- Enable "Show save confirmation" in settings
- Practice patience during slow network conditions

### Lost Formatting

**Issue:** Saved content doesn't preserve formatting

**Common causes:**
1. **Website uses complex formatting:** Notion can't represent all HTML
2. **Code blocks not detected:** Manually format as code in Notion
3. **Tables broken:** Notion tables have limitations

**Workarounds:**
- For code: Select as code before saving
- For tables: Save as text, reformat in Notion
- For images: Save separately, insert manually

## Performance Issues

**Normal save time:** 1-3 seconds
**Slow save:** 5+ seconds

**Causes & fixes:**

- Save summary instead of full text

- Check connection speed
- Use wired connection if possible
- Save drafts locally first

**Notion server load:**

4. Close unused tabs
